## 魔装連続剣

    エレメンタルスラストブレイク -> バーンストライク -> シンソイルスラスト を順番に使う
    属性は最初に確定したものを使う
    詠唱時にシンソイルスラストの線は確定
    3属性全部やる

### 炎

    属性、線確認 -> 散開 -> 頭割り -> アームズ -> サイドに避ける -> 背後で頭割り

### 雷

    属性、線確認 -> 散開 -> 散開(移動) -> サイドに避ける(広がるやつ) -> 対象者とタンク前方
    ソイル対象者はスプリント

### 光

    属性、線確認 -> 散開 -> 頭割り(3方向) -> サイドに避ける -> たけのこ捨てる -> 対象者は前面
    大爆印を他の人に当てないように
